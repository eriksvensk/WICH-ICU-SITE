HOW TO UPDATE THE WICH-ICU WEBSITE

1) Edit participating ICU sites:
   - Open data/sites.json in any text editor.
   - Add one JSON object per site with fields:
     {
       "name": "ICU name",
       "hospital": "Hospital name",
       "city": "City",
       "region": "Region/County",
       "lat": 59.3498,
       "lng": 18.0691,
       "status": "Active",
       "start": "2025-11",
       "contact": "optional email"
     }
   - Save and refresh the page. Pins and the list will update automatically.

2) Update downloadable documents:
   - Place new files in assets/docs/ (PDF or DOCX).
   - Then update data/downloads.json to include them:
     { "title": "Document title", "filename": "exact_file_name.pdf", "description": "Short description" }

3) Change homepage texts:
   - Open index.html and edit the text inside each <section>. No build step is required.

4) Colors & fonts:
   - Edit css/styles.css under :root variables.

5) Hosting:
   - This is a static site. Upload the folder to any static host (e.g., GitHub Pages, Netlify, institution web server). No backend needed.

6) Map:
   - The interactive map uses Leaflet + OpenStreetMap via CDN. Requires internet access to load map tiles.
   - If the map doesn't appear, check that data/sites.json is valid JSON (no trailing commas).
